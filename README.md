## RAG Integration

The chat UI calls the backend `/chat` endpoint defined in the companion `goodnight-gpt-svc` FastAPI project.

Setup:
1. Backend running on port 8000 (see backend README) with Pinecone + optional Anthropic configured.
2. Copy `.env.example` to `.env.local` and set:
	```
	NEXT_PUBLIC_API_BASE=http://localhost:8000
	```
3. Upload data via backend `/upload-csv` endpoint (a simple form tool can be added later) to populate the vector store before asking questions.

When an AI message is returned, top retrieved context snippets appear under the message with tooltips showing similarity scores.

If no Anthropic key is configured backend will echo plus context.
